# @amlplugins/google-cloud-error-reporting

Native `@google-cloud/error-reporting` SDK re-exported under the `@amlplugins` namespace.

> Error Reporting.

## Install

```bash
npm install @amlplugins/google-cloud-error-reporting
```

## Use

```ts
import { /* named exports from @google-cloud/error-reporting */ } from "@amlplugins/google-cloud-error-reporting";
// or
import sdk from "@amlplugins/google-cloud-error-reporting";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
